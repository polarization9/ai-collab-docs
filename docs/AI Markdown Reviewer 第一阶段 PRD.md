# AI Markdown Reviewer 第一阶段 PRD

## 1. 文档信息

- 产品名称：AI Markdown Reviewer
- 阶段名称：第一阶段，最小可运行阅读闭环
- 文档日期：2026-06-02
- 目标版本：v0.1.0
- 当前状态：已实现本地 Markdown 阅读、Mermaid 增强阅读、表格阅读增强

## 2. 阶段目标

第一阶段先让用户可以在本地打开并阅读一份 AI 生成的 Markdown / Mermaid 长文档。

用户可以通过 CLI 传入 Markdown 文件路径，系统启动本地 Web App，并在浏览器中展示文档。页面支持 Markdown 阅读、Mermaid 图表增强阅读、表格横向滚动、目录跳转和基础导出/复制能力。

第一阶段不实现批注、`.review.json` 写入、MCP server 和 Markdown 编辑，但为后续批注与 MCP 集成保留文档路径、reviewPath 和稳定 heading id。

## 3. 核心用户价值

- 本地 Markdown 文档可以快速打开，不需要复制到其他工具。
- 长文档有目录，方便跳转。
- Mermaid 图表可以直接阅读、放大、缩放、复制和导出。
- 表格在宽列和中文列场景下不会被压成不可读的一字一行。
- 阅读体验更接近一个专门面向 AI 生成文档的审阅器。

## 4. 本阶段功能范围

### 4.1 CLI 与本地服务

- 支持传入 `.md` / `.markdown` 文件路径。
- 支持相对路径和绝对路径。
- 启动本地 Express 服务。
- 默认端口 `4317`，支持 `--port` 指定端口。
- 提供 `/health` 和 `/api/document`。
- 返回文档绝对路径、相对路径、reviewPath、Markdown 原文和 headings。

### 4.2 Markdown 阅读

- 支持标题、段落、列表、引用、代码块、行内代码、链接、表格和分割线。
- 左侧目录根据服务端解析的 heading 生成。
- 点击目录可以滚动到对应章节。
- 当前章节可高亮。
- 外链新窗口打开。

### 4.3 Mermaid 阅读增强

- 识别 `language-mermaid` 代码块并渲染为 SVG。
- Mermaid 渲染结果按文档、图表序号、源码和主题缓存。
- 正文图表自动适配阅读区宽度。
- 图表点击可进入大图模式。
- Hover 工具栏支持：
  - 打开大图
  - 复制 Mermaid 源码
  - 复制 PNG
  - 导出 PNG
  - 切换亮色/暗色主题
- 每个工具按钮 hover 时显示名称气泡。
- 复制源码和复制 PNG 成功后显示右上角轻量 toast。
- 主题默认跟随系统，用户手动切换后只在 light / dark 间切换。

### 4.4 Mermaid 大图模式

- 大图模式使用遮罩 + dialog。
- 支持关闭按钮和 `Esc` 关闭。
- 支持按钮缩放、触控板捏合缩放、拖拽平移、双击重置。
- 普通双指滚动用于平移，不触发页面级缩放。
- 缩放范围支持 `1%` 到 `999%`。
- 右下角缩放比例可手输自定义值，回车或失焦后生效。
- 支持 minimap，放大后显示当前视口位置。
- 缩放和平移限制在图表边界内。

### 4.5 表格阅读增强

- Markdown 表格外层支持横向滚动。
- 表格使用固定布局，默认列宽为阅读友好的宽度。
- 表头每列右侧提供列宽拖拽手柄。
- 用户可以像表格工具一样拖拽调整列宽。
- 列宽调整在当前页面会话内生效，第一阶段不持久化。
- 手柄支持键盘左右方向键微调列宽。

## 5. 不支持范围

- 批注创建、批注保存和右侧批注线程。
- MCP server 和 MCP tool 调用。
- Markdown 编辑。
- 多文档管理。
- 表格内容编辑、冻结列、合并单元格、多选列、双击自适应列宽。
- 云同步、账号、多人协作。

## 6. 验收标准

- 执行 CLI 后可以打开本地 Markdown 文档。
- 文档目录和章节跳转正常。
- Mermaid 图表不会在滚动时反复闪烁。
- Mermaid 图表可打开大图、缩放、平移、重置。
- 大图右下角可以输入 `1-999` 的缩放比例。
- 复制源码/复制 PNG 后出现“复制成功”toast。
- Mermaid 主题切换会重新渲染图表配色，而不是只改变容器背景。
- 宽表格能横向滚动。
- 表头拖拽手柄能调整列宽，中文列不会被固定压窄。

## 7. 后续阶段预留

- 轻量 Markdown 编辑。
- 表格列宽持久化。
- 多文档打开与历史记录。
